First Program:
(Earns 150 points for PC and 20 points for Edge)

1. download bing-search-bot-edge.py
2. download MicrosoftWebDriver.exe
3. place driver in same directory as python script
4. install selenium
5. run python script

Second Program:
(Earns 100 points for mobile)

1. download bing-search-bot-mobile.py
2. set Chrome to default web browser
3. install "User-Agent Switcher for Chrome" Extension
4. switch to a mobile device using the extension
5. run python script
